# Frontend

Aplicacao React/Vite do portefolio de Artur.

## Principios

- Componentes pequenos e reutilizaveis.
- Dados centralizados em `src/utils/portfolioData.js`.
- Estilos globais divididos por variaveis, tipografia, animacoes e responsividade.
- Animacoes com Framer Motion e Intersection Observer.
- SEO, Open Graph, sitemap, robots e manifest incluidos.

## Comandos

```bash
npm install
npm run dev
npm run build
npm run preview
```
