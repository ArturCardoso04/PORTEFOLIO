# Artur Portfolio

Portefolio pessoal premium de Artur, desenvolvido com React, Vite, CSS modular e uma API Express preparada para evoluir.

## Estrutura

- `frontend/`: aplicacao React com componentes reutilizaveis, animacoes, SEO, responsividade e assets otimizados.
- `backend/`: API Express com rotas, controllers, middleware e servicos separados.

## Scripts

```bash
npm install
npm run dev
npm run dev:backend
npm run build
npm run preview
```

## Contacto

O formulario envia para `/api/contact`. Em desenvolvimento, o Vite faz proxy para `http://localhost:3001`.
