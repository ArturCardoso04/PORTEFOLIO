# Portefólio pessoal de Artur

Site one-page estático, responsivo e pronto a abrir no browser.

## Ficheiros

- `index.html`: estrutura semântica das secções Hero, Sobre Mim, Skills, Projetos, Experiência e Contactos.
- `styles.css`: sistema visual dark soft, layout responsivo, animações, hover states e menu mobile.
- `script.js`: hamburger menu, estado ativo da navegação, scroll reveal com Intersection Observer e botão de copiar resumo profissional.
- `assets/hero-bg-light.png`: imagem hero em light mode gerada para este portefólio e guardada localmente.

## Como abrir

Abre `index.html` diretamente no browser. Não é necessário instalar dependências nem correr um servidor local.

## Notas de conteúdo

Os projetos foram tratados como uma área em preparação, porque ainda não há projetos públicos fornecidos. A secção de contactos usa o email e telefone fornecidos, sem inventar GitHub, LinkedIn ou outros perfis externos.
